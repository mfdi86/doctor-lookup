# Doctor Lookup

## Description:
#### An application that finds doctors from an API

Created on 7.30.2017

## Set Up & Installation:
* Visit the following link: <https://github.com/mfdi86/doctor-lookup>

OR:

* Clone this directory onto your Desktop
* $ npm install
* $ bower install
* $ gulp build
* $ gulp serve
